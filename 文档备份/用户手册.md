# 用户手册

## 概述

TPClawHub 是公司内部托管的 AI 技能集合，包含公司所有可复用的技能（Skills），涵盖代码审查、架构设计、嵌入式开发、通信仿真等多个领域。本手册指导用户如何使用 `npx skills` 工具将这些技能一键安装到本地所有 AI IDE 中。

---

## 前置条件

### 环境要求

| 要求 | 说明 |
|------|------|
| **网络** | 必须连接公司内网（技能源仅在内网可访问） |
| **Node.js** | 需要安装 Node.js（用于运行 npx skills） |

### 检查 Node.js

```bash
# 检查是否已安装 Node.js
node --version

# 检查 npx 是否可用
npx --version
```

如果未安装 Node.js，请从 [nodejs.org](https://nodejs.org/) 下载并安装。

---

## 快速开始

### 交互式安装（推荐）

最简单的方式是直接运行命令，通过交互式界面选择要安装的技能：

```bash
# 连接到 TPClawHub 技能源
npx skills add http://192.168.62.38:3100   npx skills add http://192.168.32.62:3100
```

**执行后你会看到交互式界面**：

```
 Select skills to install: (Press <space> to select, <a> to toggle all)
❯ ◯ core-code-review        - 代码审查
  ◯ core-brainstorming       - 头脑风暴
  ◯ core-test-driven-development - 测试驱动开发
  ◯ embedded-phm-overview    - PHM开发环境搭建
  ◯ simulation-architecture  - 系统架构设计
  ... 
```
**操作说明**：
| 按键 | 功能 |
|------|------|
| `↑ / ↓` | 上下移动选择 |
| `空格` | 选中/取消选中当前技能 |
| `a` | 全选/取消全选 |
| `Enter` | 确认选择 |

**选择技能后**，会继续询问安装位置：

```
Select agents to install to: (Press <space> to select)
❯ ◯ claude-code             - Claude Code
  ◯ cursor                   - Cursor
  ◯ github-copilot           - GitHub Copilot
  ◯ windsurf                 - Windsurf
```
**最后询问**是否安装到全局目录：

```
Install to global directory? (Recommended) (Y/n)
```
建议选择 `Y`（全局安装），这样所有项目都能使用这些技能。

---

### 一键安装所有技能（自动化）

如果你想跳过交互，直接安装所有技能到所有工具：

```bash
# 安装所有技能到所有 AI 工具（全局）
npx skills add http://192.168.62.38:3100 --all -g

# 加上 -y 跳过确认提示
npx skills add http://192.168.62.38:3100 --all -g -y
```

**参数说明**：
| 参数 | 说明 |
|------|------|
| `--all` | 安装所有技能 |
| `-g` | 安装到全局目录（推荐） |
| `-y` | 跳过确认提示 |

---

### 只查看技能列表（不安装）

```bash
# 查看所有可用技能
npx skills add http://192.168.62.38:3100 --list
```

## 常用命令

> URL地址：`http://192.168.62.38:3100`

### 安装相关

```bash
# 查看可用技能
npx skills add <URL> --list

# 一键安装所有技能
npx skills add <URL> --all -g

# 安装特定技能
npx skills add <URL> --skill <skill-name>
```

### 管理相关

```bash
# 列出已安装技能
npx skills list

# 搜索技能
npx skills find <keyword>

# 检查更新
npx skills check

# 更新所有技能
npx skills update

# 卸载特定技能
npx skills remove <skill-name>

# 卸载所有技能
npx skills remove --all

# 从特定工具卸载
npx skills remove --agent claude-code <skill-name>
```
