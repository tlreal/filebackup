# Schematic Review - 原理图审查

## 概述

系统性审查电路原理图，识别潜在的设计问题。

## 审查清单

### 1. 电源部分

- [ ] 电压轨标注清晰
- [ ] 去耦电容布局合理
- [ ] 上电时序符合要求
- [ ] 电流裕量足够

### 2. MCU 部分

- [ ] 复位电路正确
- [ ] 晶振电路参数匹配
- [ ] 启动模式配置正确
- [ ] 调试接口完整

### 3. 外设接口

- [ ] 电平匹配
- [ ] ESD 保护
- [ ] 阻抗匹配（高速信号）
- [ ] 上下拉电阻

### 4. 通信接口

| 接口 | 检查项 |
|------|--------|
| UART | TX/RX 交叉连接 |
| SPI | CS/CLK/MOSI/MISO 正确 |
| I2C | 上拉电阻、地址设置 |
| CAN | 终端电阻、共地 |

### 5. 通用检查

- [ ] 网络标号一致
- [ ] 电源/地连接完整
- [ ] 未使用引脚处理
- [ ] 测试点预留

## 输出格式

```markdown
## 原理图审查报告

### 审查信息
- 项目: [项目名称]
- 版本: [原理图版本]
- 审查日期: YYYY-MM-DD

### 问题列表

| 编号 | 位置 | 级别 | 描述 | 建议 |
|------|------|------|------|------|
| 1 | U1-VDD | Critical | 缺少去耦电容 | 添加0.1uF电容 |
| 2 | J1-TX/RX | Warning | 未交叉连接 | 检查协议要求 |

### 审查结论
[通过/需修改后复审]
```
