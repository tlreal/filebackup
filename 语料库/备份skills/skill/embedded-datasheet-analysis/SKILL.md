# Datasheet Analysis - 数据手册分析

## 概述

系统性地分析 MCU 或外设数据手册，提取开发所需的关键信息。

## 适用场景

- 评估新 MCU 是否满足需求
- 开发驱动程序前理解硬件特性
- 移植代码到新平台前了解差异

## 分析框架

### 1. 基本参数

| 项目 | 需要提取的信息 |
|------|---------------|
| **内核** | 架构、主频、缓存 |
| **存储** | Flash 大小、SRAM 大小、EEPROM |
| **工作条件** | 电压范围、温度范围 |
| **封装** | 引脚数、封装类型 |

### 2. 外设能力

| 外设 | 关键参数 |
|------|---------|
| **GPIO** | 数量、复用功能、驱动能力 |
| **UART** | 数量、波特率范围、FIFO 深度 |
| **SPI** | 数量、最大时钟、主从模式 |
| **I2C** | 数量、速度模式 |
| **ADC** | 通道数、分辨率、采样率 |
| **定时器** | 数量、位宽、捕获/比较通道 |

### 3. 电气特性

| 项目 | 关键参数 |
|------|---------|
| **功耗** | 各模式电流（运行、睡眠、深度睡眠） |
| **IO 特性** | 输入输出电压阈值、驱动电流 |
| **时序** | 上电时序、复位时序 |

### 4. 开发资源

| 项目 | 需要确认 |
|------|---------|
| **开发环境** | 支持的 IDE、编译器 |
| **HAL/SDK** | 官方库版本、示例代码 |
| **调试接口** | JTAG/SWD 支持 |

## 输出格式

```yaml
芯片型号: STM32F407VGT6
分析日期: YYYY-MM-DD

基本参数:
  内核: ARM Cortex-M4F @ 168MHz
  Flash: 1MB
  SRAM: 192KB
  工作电压: 1.8V - 3.6V
  
外设资源:
  GPIO: 140 pins (多功能复用)
  UART: 6 (支持 DMA)
  SPI: 3 (支持 DMA)
  I2C: 3
  ADC: 3 x 12-bit (24通道)
  定时器: 14 (含高级定时器)
  
电气特性:
  运行电流: ~100mA @ 168MHz
  睡眠电流: ~2mA
  深度睡眠: ~10uA
  
开发资源:
  IDE: Keil MDK, STM32CubeIDE
  HAL: STM32Cube HAL
  调试: SWD, JTAG
  
关键注意事项:
  - [列出发现的重要限制或注意点]
  
与需求对比:
  - ✅ [满足的需求]
  - ⚠️ [需要注意的点]
  - ❌ [不满足的需求]
```
